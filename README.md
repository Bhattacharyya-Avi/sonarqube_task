# SonarQube Python Demo Project

A simple Python calculator project integrated with SonarQube for code quality analysis.

## Project Structure

- `src/calculator.py`: Main calculator module with arithmetic operations
- `tests/test_calculator.py`: Unit tests using pytest
- `sonar-project.properties`: SonarQube configuration
- `.github/workflows/sonarqube.yml`: GitHub Actions workflow
